# Gerenciamento de Impressoras em Rede

## Descrição
Este projeto em C tem como objetivo auxiliar a empresa **PrintTech** no gerenciamento de impressoras em diferentes departamentos de um centro de tecnologia. O sistema permite o registro de impressoras, exibição de relatórios e identificação da impressora com maior volume de impressões.

## Funcionalidades
- Cadastro de impressoras com ID, marca e total de impressões.
- Exibição da lista de todas as impressoras registradas.
- Identificação da impressora com maior número de impressões realizadas.

## Estrutura do Código
O projeto contém as seguintes funções principais:
- `cadastrar()` - Permite o cadastro de N impressoras.
- `listar()` - Exibe todas as impressoras cadastradas.
- `calculaTotalImpressao()` - Determina a impressora com maior volume de impressões.

## Melhorias Futuras

- Interface gráfica para melhor usabilidade.
- Opção para editar e remover registros de impressoras.
